
# Snake Game Deluxe

An advanced Snake Game developed using Java Swing featuring multiplayer support, dynamic levels, speed control, pause/resume system, high-score persistence, themes, obstacle mechanics, animations, sound effects, and leaderboard integration.

---

## Features

- Modern Snake Design
- Multiplayer Mode
- SQLite Leaderboard
- Sound Effects & Music
- Difficulty Levels
- Theme Switching
- Wrap Around Mode
- Obstacle System
- High Score Saving
- Smooth Animations

---

## Technologies Used

- Java
- Swing
- AWT
- SQLite
- OOP
- File Handling
- Event Handling

---

## Controls

| Key | Action |
|---|---|
| Arrow Keys | Move |
| P | Pause |
| ENTER | Restart |
| + / - | Speed Control |
| T | Theme Switch |
| W | Wrap Mode |

---

## How To Run

### Compile

```bash
javac SnakeGame.java
```

### Run

```bash
java SnakeGame
```
